# precificacao-mercado-livre

# 💰 Sistema de Precificação - Mercado Livre

## 📌 Descrição
Aplicação em C desenvolvida para automatizar o cálculo de preços de produtos no Mercado Livre.

## Tecnologias
- Linguagem C

## Funcionalidade

Processamento de planilhas com custos de produtos
Cálculo automatizado de precificação considerando regras de negócio
Geração de saída estruturada com preços para modalidades premium e clássica

## 💡 Contexto
Projeto criado para otimizar o processo de precificação em ambiente empresarial, reduzindo erros e aumentando a eficiência.
